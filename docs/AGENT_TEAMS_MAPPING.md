# Agent Teams Mapping (Claude concept adapted)

Reference: https://code.claude.com/docs/en/agent-teams

## Team members mapped from your current bots
- COO/Orchestrator: 씽패
- PM Lead: 기획실장
- Frontend: 픽셀요정
- Backend Main: 백엔드 개발자 메인
- Backend Sub/Infra: 백엔드 개발자 서브
- Designer: 디자이너
- Marketing: 마케터
- BizOps: 비즈맨
- Research: 리서치맨

## Team protocol
- CEO request -> Orchestrator triage
- role-specific execution in dedicated channels
- decisions/tasks persisted to markdown + json
- major conflicts escalated to COO + PM
